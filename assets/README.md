# Carpeta de recursos (assets)

## ⚠️ Falta agregar el logo

Para que el **favicon** (ícono de la pestaña) y la **vista previa al compartir en redes**
(Open Graph) muestren el logo real de IDANCE, guarda el logo aquí con este nombre exacto:

```
assets/logo.jpg
```

- Usa el logo cuadrado sobre fondo negro (el que aparece en Facebook/Instagram).
- Formato `.jpg` o cámbialo a `.png` y actualiza las referencias en `index.html`
  (`assets/logo.jpg` → `assets/logo.png`).
- Tamaño recomendado: 1080×1080 px (cuadrado).

Mientras no exista el archivo, la página usa un ícono de respaldo (emoji 💃) y todo
lo demás funciona con normalidad — el logotipo del sitio está dibujado con tipografía
y colores de la marca, así que no se ve roto.
