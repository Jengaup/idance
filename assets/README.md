# Carpeta de recursos (assets)

## Logo

El logo oficial de IDANCE está en:

```
assets/Idancelogo.jpg   (960×960, cuadrado, fondo negro)
```

Se usa como **favicon** (ícono de la pestaña), en la **vista previa al compartir
en redes** (Open Graph) y en el **footer** del sitio.

Si en el futuro reemplazas el archivo, mantén el mismo nombre (`Idancelogo.jpg`)
o actualiza las referencias en `index.html`. Como respaldo del favicon, la página
usa un emoji (💃) si el archivo no estuviera disponible.
